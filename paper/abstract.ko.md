# Abstract (한국어)

> 상태: 초안 v1. 매트릭스 데이터 수집(마일스톤 M4)과 논문 framing 확정에
> 따라 수정 예정. 플레이스홀더 `N`, `M`은 벤치마크 매트릭스 수집 후 채움.

LLM 에이전트 시스템은 프롬프트, 도구, 모델, 오케스트레이션 구성이 얽힌
다차원 변수 공간을 가지며, 이로 인해 체계적 실험과 공정한 비교가 어렵다.
기존 도구들은 응용 프레임워크(LangChain), 프로덕션 모니터링(LangSmith),
정적 벤치마크(SWE-bench) 등 인접 영역을 다루지만, 변수를 독립적으로 분리해
실험하는 데 특화된 플랫폼은 부재하다. 본 연구는 이러한 공백을 메우는
**nature**를 제안한다 — Pack / Host / Agent / Preset 아키텍처와 이벤트 소싱
기반 실행을 통해 각 변수를 합성 가능하고 분리 가능한 단위로 다루어, 재현
가능한 실험과 공유 가능한 연구 산출물을 지원한다. 전체 세션 단위 비교를
넘어, nature의 이벤트 소싱 런타임은 **event-pinned counterfactual**을 제공한다
— 기록된 세션을 임의의 결정 지점에서 포크하고, 변형된 구성 아래에서 이어
진행하여, 관찰된 delta를 분기 이후 바뀐 변수만으로 깨끗하게 귀속시킬 수
있게 한다. `N`개 태스크 × `M`개 구성 × 3 시드의 preset 단위 벤치마크를
통해 구성 변화가 단일 모델 선택보다 더 큰 비용-지연 분산을 유발함을
보이며, 동일 히스토리에서 분기한 event-pinned ablation으로 프롬프트·모델
라우팅 차원을 분리해 보여준다. 파일 기반 스키마로
산출물을 공유 가능하게 설계함으로써, 효과적인 agent 구성에 관한 지식을
커뮤니티 차원에서 축적하는 기반을 목표로 한다.
